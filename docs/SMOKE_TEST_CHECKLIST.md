# AlgoSprint 스모크 테스트 체크리스트

## 사전 준비
- `npm install`
- `npx prisma generate && npx prisma db push`
- `npm run dev`

## 핵심 라우트
- [ ] `/auth/signin` 한국어 로그인 문구/버튼 확인
- [ ] `/` 대시보드 한국어 지표/버튼 확인
- [ ] `/problems` 문제 목록 표시, 태그/난이도 표시 확인
- [ ] `/problems/[id]` 문제 설명 + 개념 카드 4개 섹션 노출 확인
- [ ] `/submissions` 제출 내역 한국어 문구 확인

## 인증/런타임
- [ ] 개발 계정 로그인 가능
- [ ] 문제 상세 페이지 비로그인 접근 시 로그인 페이지 리다이렉트
- [ ] 코드 작성 후 임시저장(draft) 문구/동기화 시간 표시 확인

## 채점 워크플로
- [ ] 컴파일/실행/디버그/제출 버튼 동작
- [ ] 출력 탭 전환 정상 동작
- [ ] 제출 후 히스토리에 상태/테스트케이스 표시

## 배포 전 체크
- [ ] `npm run lint`
- [ ] `npm run build`
